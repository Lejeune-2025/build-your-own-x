# Dossier `formation/`

Chaque sous-dossier = un projet documenté dans **Fruits de ma formation** (README racine).

```text
formation/
  mini-shell/
    README.md
    capture.png
  ray-tracer/
    rendu.png
```

Référence les images depuis le `README.md` à la racine avec un chemin relatif, par exemple :

- `![Capture du terminal](formation/mini-shell/capture.png)`
- `![Rendu final](formation/ray-tracer/rendu.png)`
