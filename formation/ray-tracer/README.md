# Ray tracer

Exemple de structure (souvent en C++ pour le tuto *Ray Tracing in One Weekend*) :

- `main.cc` — image PPM en sortie, boucle sur les pixels.
- `vec3.h`, `ray.h` — maths de base.
- `sphere.h`, `hittable_list.h` — géométrie et tests d’intersection.

Ajoute `rendu.png` ici (export de ton image PPM ou capture du fichier ouvert) pour l’affichage dans le README racine.
